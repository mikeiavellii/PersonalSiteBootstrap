Library Terminal
January 2023

This was my first group project/midterm in my C# Fullstack Bootcamp. In this project we took user input and allowed the to use different options to search, check out, and return a list of books. While those were the requirements, my group took it upon themselves to make a whole user system and that held their own individual picks specific to them as well as few other things.

This gave me an entirely knew perspective. Being our first group project. It was our first time handling merge issues and other things as well. All and all it was a wonderful little console app.
